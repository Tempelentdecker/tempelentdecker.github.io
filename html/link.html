Anleitung: Interne Links in HTML

Hier ist ein einfacher HTML-Code, der eine interne Verlinkung demonstriert. Der anklickbare Text führt zur "meineSeite.html", die sich im gleichen Verzeichnis wie die Hauptseite befinden sollte:

<!DOCTYPE html>
<html>
    <head>
        <title>Meine Webseite</title>
    </head>
    <body>
        <h1>Willkommen auf meiner Webseite</h1>
        <p>Das ist die Hauptseite meiner Website. Für mehr Informationen, <a href="meineSeite.html">klicken Sie hier</a>.</p>
    </body>
</html>

In diesem Beispiel ist "klicken Sie hier" der anklickbare Text, der auf die interne Seite "meineSeite.html" verweist. Wenn ein Benutzer darauf klickt, wird er zur "meineSeite.html" weitergeleitet. Bitte stelle sicher, dass die "meineSeite.html" im gleichen Verzeichnis wie diese Hauptseite existiert.

Diese Technik ist sehr nützlich, wenn du mehrere Seiten auf deiner Website hast und du zwischen diesen navigieren möchtest.
